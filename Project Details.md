# Disaster Alert System

## Project Title
Disaster Alert System

## Abstract

The Disaster Alert System is designed to notify users about natural disasters quickly and efficiently. The system provides alerts, emergency contacts, and safety precautions through a simple web interface. The main goal is to reduce risk by spreading important disaster information on time.

## Software Requirements

- Visual Studio Code
- XAMPP
- PHP
- MySQL
- Google Chrome

## Hardware Requirements

- Intel i3 Processor or above
- 4GB RAM
- 100GB Storage

## Modules

1. User Registration
2. User Login
3. Dashboard
4. Disaster Alert Module
5. Emergency Contact Module
6. Admin Module

## Future Enhancement

- Live Weather API
- SMS Alerts
- Email Notifications
- GPS Tracking
- Mobile Application

## Conclusion

The Disaster Alert System improves disaster awareness by providing timely alerts and emergency information to users.
