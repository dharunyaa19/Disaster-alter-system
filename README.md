# 🌍 Disaster Alert System

## 📌 Project Description
The Disaster Alert System is a web-based application developed to provide timely disaster alerts and emergency information to users. It helps people stay informed about natural disasters such as floods, cyclones, earthquakes, and heavy rainfall, enabling them to take necessary precautions.

---

## 🎯 Objectives
- Provide real-time disaster alerts.
- Display safety guidelines.
- Show emergency contact numbers.
- Allow user registration and login.
- Improve disaster preparedness.

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

---

## ✨ Features
- User Registration
- User Login
- Home Dashboard
- Disaster Alerts
- Emergency Contacts
- Safety Tips
- Admin Panel
